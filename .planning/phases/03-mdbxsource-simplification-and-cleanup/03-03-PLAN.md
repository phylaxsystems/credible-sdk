Two steps:

**Step 1 — Remove the [[bin]] section from crates/state-worker/Cargo.toml.**

Delete the entire `[[bin]]` block:
```toml
[[bin]]
name = "state-worker"
path = "src/main.rs"
```

The `[lib]` section stays untouched:
```toml
[lib]
name = "state_worker"
path = "src/lib.rs"
```

Also remove any dependencies that are ONLY used by main.rs and NOT by any lib source file.
To determine what to remove, check the imports in main.rs:
- `clap` — used only in `cli.rs` (which is declared `mod cli;` in main.rs). If cli.rs is only
  reached from main.rs (not from lib.rs), remove `clap` from [dependencies].
- `rust-tracing` — used in main.rs for `trace()`. Check if lib.rs or any lib module uses it.
  If not, remove.
- `rustls` — used in main.rs for crypto provider. Check lib usage. If not used in lib, remove.
- `futures-util` — used in main.rs for `.catch_unwind().await`. Check lib usage.
- `tokio::sync::broadcast` — used in main.rs shutdown signal. Check lib usage.

Safe rule: only remove a dependency if it appears in main.rs imports but NOT in lib.rs or any
module file that lib.rs declares. When in doubt, KEEP the dependency — a spurious dependency
causes no compilation error, but a missing one does.
